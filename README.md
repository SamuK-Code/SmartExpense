# 💳 Credit Card Tracker v2.1

App completo para monitorar gastos do cartão de crédito com bancos brasileiros oficiais.

## ✨ Funcionalidades

- 🏦 **30+ Bancos Brasileiros** - Lista pré-pronta com Nubank, Inter, Itaú, Bradesco, etc.
- 🌙 **Modo Escuro** - Alternância suave entre temas
- 💳 **Múltiplos Cartões** - Cadastre vários cartões com limites
- 🔔 **Alertas Inteligentes** - Pop-ups quando ultrapassar limites
- 📅 **Filtro por Período** - Hoje, 7 dias, Mês, Ano
- ✏️ **Editar Gastos** - Toque para editar
- 📊 **Gráficos Detalhados** - Por categoria, mês e cartão
- ✨ **Animações Suaves** - Reanimated 3
- 📤 **Exportar CSV** - Para planilha

## 🏦 Bancos Suportados

### Digitais
Nubank, Inter, C6 Bank, Next, Neon, PicPay, Will Bank, Digio, Banco Original, Banqi

### Tradicionais
Itaú, Bradesco, Santander, Banco do Brasil, Caixa, Safra, Banco Pan, Citibank, HSBC

### Cartões
American Express, Hipercard, Aura

### Fintechs
Mercado Pago, PagBank, Stone, Cielo, Rede, Ame Digital, RecargaPay

## 🚀 Como rodar

```bash
cd credit-card-tracker-v2
npm install
npx expo start
```

Escaneie o QR code com o Expo Go!
